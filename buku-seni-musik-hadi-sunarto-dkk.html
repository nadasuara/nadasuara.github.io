10. Harga 2buku tulis 3 buku gambar dan 3 pensil Rp15,000.00 perbandingan harga buku tulis buku gambar dan pensil =5 : 3 : 2 .Jika hadi membeli 3 buku tulis 1 buku gambar dan 2 pensil hadi harus membayar


Penjelasan

Jika Buku Tulis = x ; Buku Gambar = y ; Pensil = z, maka :

2x + 3y + 3z = 15.000

x : y : z = 5 : 3 : 2

2x + 3y + 3z = (2 × 5) + (3 × 3) + (3 × 2) = 25 banding = 15.000

3x + 1y + 2z = (3 × 5) + (1 × 3) + (2 × 2) = 22 banding = A

A = 22 × 15.000 ÷ 25

A = Rp13,200.00

14. harga 2 buku tulis 3 buku gambar dan 3 pensil rp 15000.perbandingan buku tulis:buku gambar:pensil=5:3:2.hadi membeli 3 buku tulis 1 buku gambar 2 pensil . hadi harus membayar....​


Buku tulis (t)
Buku gambar (g)
Pensil (p)

2t + 3g + 3p = 15.000
t : g : p = 5 : 3 : 2

3t + 1g + 2p = ?

t/g = 5/3
t = 5g/3

g/p = 3/2
p = 2g/3

2t + 3g + 3p = 15.000
2(5g/3) + 3g + 3(2g/3) = 15.000
10g/3 + 3g + 6g/3 = 15.000
10g + 9g + 6g = 45.000
25g = 45.000
g = 1.800

t = 5g/3
= 5(1.800)/3
= 3.000

p = 2g/3
= 2(1.800)/3
= 1.200

Maka 3t + 1g + 2p
= 3(3.000) + 1(1.800) + 2(1.200)
= 9.000 + 1.800 + 2.400
= 13.200

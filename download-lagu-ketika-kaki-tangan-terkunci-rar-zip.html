Jawaban:

1.Pastikan Anda telah masuk ke akun Repl.it yang benar. Pastikan Anda sudah login dan mengakses repl tersebut dengan akun yang tepat.

2.Periksa kembali URL repl yang ingin Anda unduh sebagai file zip. Pastikan URL tersebut valid dan repl yang ingin Anda unduh masih ada.

2.Refresh atau muat ulang halaman. Terkadang, masalah ini bisa diselesaikan dengan memuat ulang halaman atau membuka repl di tab yang baru.

4.Coba gunakan browser yang berbeda. Jika Anda menggunakan browser yang sama dan mengalami masalah, cobalah menggunakan browser yang berbeda untuk melihat apakah masalahnya masih ada.

5.Ajukan pertanyaan di komunitas Repl.it. Jika semua langkah di atas tidak berhasil, Anda dapat mencoba mengajukan pertanyaan atau masalah yang Anda hadapi ke komunitas Repl.it. Banyak pengguna Repl.it yang berpengalaman di sana dapat membantu Anda menemukan solusi yang sesuai.

Jika masalah masih belum teratasi setelah mencoba langkah-langkah di atas, disarankan untuk menghubungi tim dukungan Repl.it langsung untuk mendapatkan bantuan lebih lanjut.

36. Di ketahui suatu server (debian), memiliki suatu file berkestensi .zip dan .rar, agar file tersebut bisa digunakan dan di pindahkan ke folder root /var/www/html langkah apa yang harus di lakukan oleh administrator server tersebut?​


Langkah yang harus di lakukan oleh administrator server tersebut adalah dengan cara mengobah alamat menjadi /home/(nama_user)/public_html/>. Lalu setelah itu lakukan sudo restart.

Pembahasan

Server adalah sistem komputer yang memiliki layanan khusus berupa penyimpanan data. Peran server sangat penting untuk mengirim atau menerima data dan informasi yang tersedia.Pada umumnya tugas utama komputer server adalah melayani dan bertanggung jawab penuh atas permintaan data dari komputer client. Ini juga berperan sebagai pengatur hak akses jaringan yang dapat digunakan komputer klien. Server adalah sistem komputer yang menyediakan sumber daya untuk menyimpan data. Biasanya, penyimpanan data di server digunakan untuk menyimpan dokumen dan data yang digunakan untuk melakukan berbagai layanan. Misalnya melihat website, mengirim email dan lain-lain.

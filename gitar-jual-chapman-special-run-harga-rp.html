17. Enam belas gitar dibeli seharga Rp.95.000,00 per buah dengan ongkos kirim seluruhnya Rp.88.000,00. Jika dijual dengan keuntungan Rp.15.000,00 per buah, maka harga penjualan gitar per buah sebesar.../tolong dijawab


95.000 + ((88.000 :16)+15.000= n

n =95.000 + 5.500 + 15.000
= 115.500
jadii gitar perbuahnya dijual seharga Rp.115.500,00

terimakasihhAritmatika Sosial

Harga pembelian 16 gitar Rp 95.000 / buah
Ongkos Kirim Rp 88.000,-
Keuntungan Rp 15.000 / buah.

Harga penjualan gitar per buah adalah...

Harga penjualan = Harga pembelian + ongkos kirim + keuntungan
= (Rp 95.000 x 16) + 88.000 + (15.000 x 16 )
= Rp 1.520.000 + 88.000 + 240.000
= Rp 1.848.000,-
= Rp 115.500,- ( per buah )

Jawaban:

A. Untung

B. Rp. 1.500.000

C. 10%

Penjelasan dengan langkah-langkah:

A. Rugi atau untung ?

Untung

Karena Harga beli = 15.000.000

Dan harga jual = 16.500.000

Maka

16.500.000 - 15.000.000

= 1.500.000

Jadi Untung sebesar Rp. 1.500.000

B. Berapakah harga rugi/untung yang diperoleh?

Harga untung

Rp. 16.500.000 - Rp. 15.000.000

= Rp. 1.500.000

Harga untung nya adalah sebesar Rp. 1.500.000

C. Cara mencari presentase untung =

[tex] = \frac{1.500.000}{15.000.000} \times 100[/tex]

[tex] \frac{1.500.000}{150.000} = \frac{150}{15} = 10[/tex]

[tex] = 10\%[/tex]

Jadi presentase untung yang diperoleh adalah 10%

Mengapa pak Dlean mengalami keuntungan?

Karena harga jual motor pak Dlean nilainya lebih tinggi daripada harga belinya, sehingga dalam hal ini Pak Dlean mendapatkan pendapatan lebih dari penjualan motornya.

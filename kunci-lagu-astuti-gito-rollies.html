27. Perbandingan Uang Gito dengan Gita adalah 3:4 . Jika Gita memberikan Rp25.000 kepada Gito. Uang mereka akan sama. Berapa uang keduanya masing masing?


Bab Perbandingan
Matematika SD Kelas VI

Gito = a
Gita = b

a : b = 3 : 4
a = 3b/4

a + 25.000 = b - 25.000
3b/4 + 25.000 = b - 25.000
3b + 100.000 = 4 (b - 25.000)
3b + 100.000 = 4b - 100.000
4b - 3b = 100.000 + 100.000
b = 200.000

a = 3b/4
a = 3 x 200.000 / 4
a = 600.000/4
a = 150.000

jadi, uang Gito = Rp 150.000,00
        uang Gita = Rp 200.000,00Uang Gito = x
Uang Gita = y

x : y = 3 : 4
x = 3y/4

y - 25000 = x + 25000
y - 25000 - 25000 = 3y/4
y - 3/4 y = 50000
1/4 y = 50000
y = 4 × 50000
y = uang Gita = 200000

x = 3/4 y
x = 3/4 × 200000
x = uang Gito = 150000

Jadi, uang mreka masing-masing :
Uang Gito = 150000
Uang Gita = 200000

Afgan = A
Deri = D

S : A = 2 : 3
A : D = 6 : 5

Satukan perbandingan antara S, A, dan D

S : A : D = (2x2) : 6 : 5 = 4 : 6 : 5

misalkan banyak uang dengan X

S + A = D + 200.000
4X + 6X = 5X + 200.000
10X = 5X + 200.000
10X - 5X = 200.000
5X = 200.000
X = 40.000

Uang Deri = 5X = 5 x 40.000 = 200.000

Mata Pelajaran: Teknologi Informatika
Kata kunci: Pseudocode

• Jawaban pendek:

Keterangan soal:
1. Inisialisasi variabel jarak, waktu, kecepatan
2. Jarak = 5000
3. Jika waktu 0, maka kecepatan = 0
4. Jika tidak, maka kecepatan = jarak / waktu
5. print kecepatan

Pseudocode:
- Definisi
jarak, waktu, kecepatan : integer

- Deskripsi
jarak <= 5000
print ("Masukkan waktu: ")
read(waktu)
if waktu = 0 then
  kecepatan <= 0
else
  kecepatan <= jarak / waktu
end if
print (kecepatan)

• Penjelasan:
Variabel jara, waktu, dan kecepatan diinisialisasi sebagai data integer agar dapat melakukan perhitungan.

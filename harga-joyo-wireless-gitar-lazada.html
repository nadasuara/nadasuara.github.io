21. Joyo dan Budi masing-masing memiliki uang yang jika di gabung total ya menjadi 1.juta rupiah perbandingan uang Joyo dan Budi adalah 2:3 jika Joyo memperoleh tambahan 300 ribu, maka berapa perbandingan uang mereka sekarang


Jawaban :

Jawaban yang benar adalah 7 : 6

Pembahasan :Misalkan :

j = besar uang Joyo

b = besar uang Budi

j + b = 1.000.000 ... Persamaan 1

j : b = 2 : 3

3j = 2b

3j - 2b = 0 ... Persamaan 2

Eliminasi b pada persamaan 1 dan 2 :

j + b = 1.000.000 (dikali 2)

3j - 2b = 0 (dikali 1)

2j + 2b = 2.000.000

3j - 2b = 0

_________________ +

5j = 2.000.000

j = 2.000.000/5

j = 400.000

Substitusi j = 400.000 ke persamaan 1 :

j + b = 1.000.000

400.000 + b = 1.000.000

b = 1.000.000 - 400.000

b = 600.000

Uang Joyo = 400.000

Joyo memperoleh tambahan uang Rp300.000,00, sehingga uang Joyo menjadi :

Total uang Joyo = 400.000 + 300.000 = 700.000

Uang budi = 600.000

Perbandingan Uang Joyo : Uang budi = 700.000 : 600.000

= 7 : 6

Jadi perbandingan uang mereka sekarang 7 : 6 .

tujuh puluh satu maaf kalo salahayam = x
kambing = y

x = 25 - y

50 - 2y + 4y = 62
2y = 62 - 50
y = 12/2

x = 25 - 6
